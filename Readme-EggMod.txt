EggMod is a simple example mod. It's main purpose is to show you how to mod The Guild 3. The mod also includes a PDF with the script documentation for The Guild 3. You can find the PDF in the main folder of the game!
Nevertheless, EggMod brings some nice features into the game. Have fun! :-)

Please note: In addition to English texts for all features in this mod, there is (so far) only a German translation!

New products/items:
- Chicken: Can be raised at the farm
- Eggs: Can be gathered from chicken at the farm
- Fried chicken: Can be cooked from chicken in the residence (Consumable)
- Boiled egg: Can be cooked from eggs in the residence (Consumable)
- Tame chicken: Children can try to hatch a tame chicken out of an egg...

More storage slots:
- Farm: +2 Slots
- all Residences: +2 Slots

New actions:
- Hand over item (see below)
- Consume boiled egg / fried chicken
- Hatch chicken
